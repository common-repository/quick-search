Quick Search add AJAX Search to your site

== Description ==

Quick Search add AJAX Search to your site sorting results by Posts, Page and Comments.
You can easly customize the popup menu style by modifying the file quick-search.css.
You can find some option through the 'Plugins' menu in WordPress.

== Installation ==

1. Upload `quick-search` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
